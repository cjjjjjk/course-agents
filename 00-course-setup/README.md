# Course Orientation

This reading edition of the course does not require a local Python environment, Azure subscription, Jupyter, or .NET SDK. The lessons are published as static pages for studying AI agent concepts, design patterns, and deployment methods.

## How to Use This Course

1. Start with [Intro to AI Agents and Agent Use Cases](../01-intro-to-ai-agents/README.md) if you are new to agentic systems.
2. Continue through the numbered lessons for a structured learning path.
3. Use the production, protocol, memory, browser-use, and security lessons as reference material when designing real systems.
4. Treat code blocks as conceptual examples unless a lesson explicitly frames them as pseudocode or architecture notes.

## Recommended Reading Path

| Stage | Lessons | Focus |
| --- | --- | --- |
| Foundations | 01, 02, 03 | Agent concepts, frameworks, and design patterns |
| Core capabilities | 04, 05, 06, 07, 08, 09 | Tool use, RAG, trust, planning, multi-agent systems, metacognition |
| Production methods | 10, 11, 12, 13, 14, 15, 18 | Observability, protocols, context, memory, framework concepts, computer use, security |
