# Định hướng khóa học

Bản đọc của khóa học này không yêu cầu môi trường Python cục bộ, Azure subscription, Jupyter hoặc .NET SDK. Các bài học được xuất bản dưới dạng trang tĩnh để nghiên cứu khái niệm AI agent, mẫu thiết kế và phương pháp triển khai.

## Cách học

1. Bắt đầu với [Giới thiệu về AI Agents và các trường hợp sử dụng](../01-intro-to-ai-agents/README.md) nếu bạn mới học hệ thống agentic.
2. Đi tiếp theo thứ tự bài học để có lộ trình có cấu trúc.
3. Dùng các bài production, protocol, memory, browser-use và security như tài liệu tham khảo khi thiết kế hệ thống thật.
4. Xem code block là ví dụ khái niệm, trừ khi bài học ghi rõ đó là pseudocode hoặc ghi chú kiến trúc.

## Lộ trình đọc đề xuất

| Giai đoạn | Bài | Trọng tâm |
| --- | --- | --- |
| Nền tảng | 01, 02, 03 | Khái niệm agent, framework và mẫu thiết kế |
